## Prerequisites

Before you begin, ensure you have:

-   A Teamm center account with API access enabled
-   API keys from your center's settings
-   Basic understanding of REST APIs

## Quick Start

### 1. Obtain API Keys

1. Navigate to **General Settings** â†’ **Other Settings** in your Teamm center
2. Click on **API Secret Keys**
3. Create a new application and note down your keys:
    - App ID
    - Public Key
    - Secret Key

### 2. Make Your First Request

Here's a simple example to get started:

<CodeGroup>

```bash cURL
curl -X GET "https://api.teamm.com/v1/guests" \
  -H "X-API-APP-ID: your-app-id" \
  -H "X-API-PUBLIC-KEY: your-public-key" \
  -H "X-API-SECRET-KEY: your-secret-key" \
  -H "X-LOCALE: en" \
  -G -d "startDate=2025-01-01" \
     -d "endDate=2025-01-31" \
     -d "start=1" \
     -d "length=10"
```

```javascript JavaScript
const response = await fetch(
	'https://api.teamm.com/v1/guests?' +
		new URLSearchParams({
			startDate: '2025-01-01',
			endDate: '2025-01-31',
			start: '1',
			length: '10',
		}),
	{
		headers: {
			'X-API-APP-ID': 'your-app-id',
			'X-API-PUBLIC-KEY': 'your-public-key',
			'X-API-SECRET-KEY': 'your-secret-key',
			'X-LOCALE': 'en',
		},
	}
);

const data = await response.json();
console.log(data);
```

```python Python
import requests

url = "https://api.teamm.com/v1/guests"
headers = {
    "X-API-APP-ID": "your-app-id",
    "X-API-PUBLIC-KEY": "your-public-key",
    "X-API-SECRET-KEY": "your-secret-key",
    "X-LOCALE": "en"
}
params = {
    "startDate": "2025-01-01",
    "endDate": "2025-01-31",
    "start": 1,
    "length": 10
}

response = requests.get(url, headers=headers, params=params)
data = response.json()
print(data)
```

</CodeGroup>

## Common Integration Patterns

### Sync Guest Data

```javascript
// Poll for new/updated guests every hour
setInterval(async () => {
	const guests = await fetchGuests({
		startDate: new Date().toISOString().split('T')[0],
		endDate: new Date().toISOString().split('T')[0],
	});

	// Process new guests
	guests.data.forEach((guest) => {
		updateLocalDatabase(guest);
	});
}, 3600000);
```

### Handle Pagination

```javascript
async function getAllGuests(startDate, endDate) {
	let allGuests = [];
	let currentPage = 1;
	let totalPages = 1;

	do {
		const response = await fetchGuests({
			startDate,
			endDate,
			start: currentPage,
			length: 50,
		});

		allGuests = allGuests.concat(response.data);
		totalPages = Math.ceil(response.total / 50);
		currentPage++;
	} while (currentPage <= totalPages);

	return allGuests;
}
```

## Error Handling

Always handle potential errors in your API calls:

```javascript
try {
	const response = await fetch(url, { headers });

	if (!response.ok) {
		throw new Error(`HTTP ${response.status}: ${response.statusText}`);
	}

	const data = await response.json();
	return data;
} catch (error) {
	console.error('API Error:', error.message);
	// Handle error appropriately
}
```
