---
title: 'GET /sessions/list'
description: 'Get a list of sessions within a specified date range.'
---

<Snippet file="humanized-format-note" />

#### Authentication

<Snippet file="authentication-info" />

#### Query Params

<ParamField path="startDate" type="Date" required>
	Filter by start date (from this date). Format: dateIsoString
</ParamField>

<ParamField path="endDate" type="Date" required>
	Filter by end date (to this date). Format: dateIsoString
</ParamField>

<Snippet file="pagination-params" />

#### Example Request

```bash
GET /sessions/list?startDate=2025-01-01&endDate=2025-01-31&start=1&length=25
```
